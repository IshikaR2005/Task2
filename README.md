⏱️ Precision Stopwatch

A modern and responsive Stopwatch Web Application built using HTML, CSS, and JavaScript.
This project provides accurate time tracking with features like Start, Pause, Reset, and Lap recording, along with highlighting the fastest and slowest lap times.

This project was developed as part of my Web Development Task 02.

🚀 Features

▶️ Start Timer – Begin the stopwatch.

⏸️ Pause Timer – Pause the running stopwatch.

🔄 Reset Timer – Reset the stopwatch to zero.

🏁 Lap Functionality – Record lap times while the stopwatch runs.

⚡ Fastest Lap Detection – Automatically highlights the fastest lap.

🐢 Slowest Lap Detection – Automatically highlights the slowest lap.

🗑️ Clear All Laps – Remove all recorded laps.

🎨 Modern Dark UI – Clean and stylish user interface.

🛠️ Technologies Used

HTML5 – Structure of the web application

CSS3 – Styling and UI design

JavaScript (Vanilla JS) – Stopwatch logic and interactivity
